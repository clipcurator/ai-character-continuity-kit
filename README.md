# AI Character Continuity Kit

AI character continuity is the practice of keeping a character recognizable across scripts, scenes, storyboard frames, generated images, and video shots. This repository collects templates and checklists for character bibles, visual consistency, relationship mapping, and short drama production.

It is designed for AI film, vertical drama, micro-drama, storyboard, and script-to-video workflows.

## Why character continuity matters

In AI-assisted drama production, character drift can happen at many points:

- The face changes between scenes.
- Clothing or hairstyle changes without story reason.
- A character's personality becomes inconsistent.
- Relationship dynamics are forgotten across episodes.
- Visual prompts describe the same person differently.

A continuity kit gives the production team a shared source of truth before generating storyboards, scenes, or video.

## Continuity dimensions

| Dimension | What to define | Why it matters |
|---|---|---|
| Identity | Name, age range, role, archetype | Keeps the character recognizable in story logic |
| Visual design | Face, hair, body type, clothing, palette | Reduces visual drift across generated assets |
| Personality | Voice, emotion range, decision style | Keeps behavior consistent |
| Relationship map | Allies, rivals, family, romance | Preserves drama tension |
| Episode memory | Past choices, secrets, injuries, promises | Maintains continuity across episodes |

## Production workflow

1. Create a character bible before generating visuals.
2. Define stable visual anchors: face, silhouette, clothing, color palette, and signature props.
3. Define flexible elements: expressions, action poses, lighting, and scene-specific wardrobe.
4. Keep a relationship map for recurring characters.
5. Review every storyboard or generated scene against the continuity checklist.
6. Update the bible only when the story intentionally changes the character.
