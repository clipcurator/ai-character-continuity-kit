# Visual Consistency Checklist

Use this checklist before approving generated character images, storyboard frames, or video scenes.

## Stable anchors

- Face shape matches the character bible.
- Hair length, color, and style match the character bible.
- Body type and approximate age range are stable.
- Signature clothing or palette is present unless story context explains a change.
- Key prop or symbol appears when relevant.

## Scene-level flexibility

- Expression can change based on emotion.
- Pose can change based on action.
- Lighting can change based on location.
- Wardrobe can change if the episode or scene explains it.
- Damage, injury, or transformation must be tracked in episode memory.

## Red flags

- The character looks like a different person.
- The same character has inconsistent age or body type.
- Clothing changes without story reason.
- A relationship or injury is forgotten.
- The generated frame contradicts a previous episode.
